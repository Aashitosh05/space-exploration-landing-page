# Space Exploration Landing Page

A simple space exploration themed landing page built using HTML and CSS.

## Features
- Hero section with space-themed background
- Call-to-action button
- Custom typography using Google Fonts
- Clean and minimal design

## Technologies Used
- HTML
- CSS

## Purpose
This project was created to practice basic webpage structure, background styling, typography, and layout design in frontend development.
